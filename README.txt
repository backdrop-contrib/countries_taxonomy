DESCRIPTION
------------
This module provides countries taxonomy list.

REQUIREMENTS
------------
Drupal 7.x

INSTALLATION
------------
1.  Place countries_taxonomy modules into your modules directory.
    This is normally the "sites/all/modules" directory.

2.  Go to admin/build/modules. Enable modules.
    The Countries Taxonomy modules is found in the Taxonomy section.
